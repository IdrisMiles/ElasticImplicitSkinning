_fieldFuncs: Device pointer to field function 3D textures.

_centroidWeights: Device pointer to one ring centroid weights calculated using MVC.
_neighScatterAddr: Device pointer holding start index into _oneRingVerts for each vertex.
_sigma: scaling value for vertex projection step.
_contactAngle: threshold for running vertex projection step
_iterations: max number of iterations of vertex projection and tengential relaxations.
